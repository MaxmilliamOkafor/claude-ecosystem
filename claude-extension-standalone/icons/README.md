# Extension icons

Drop `icon-16.png`, `icon-32.png`, `icon-48.png`, and `icon-128.png` here
before running `pnpm build`. Any square PNG works.

If you want to reuse the Anthropic glyph, copy `icon-128.png` from
https://github.com/MaxmilliamOkafor/Download-Claude-Extension and resize
to the smaller sizes with ImageMagick, an online resizer, or Figma.
